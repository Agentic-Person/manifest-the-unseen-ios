# Placeholder Assets

These are temporary placeholder images for development with Expo Go.

## Current Placeholders

- **icon.png** (1024x1024) - App icon placeholder (purple with "M" text)
- **splash.png** (1242x2436) - Splash screen placeholder (purple with app name)
- **adaptive-icon.png** (1024x1024) - Android adaptive icon placeholder

## Replace Before Production

Before building for TestFlight or App Store, replace these with:
1. Professional app icon (1024x1024 PNG, no transparency for iOS)
2. Splash screen design matching brand
3. Android adaptive icon (foreground + background)

## How to Generate Proper Assets

Use tools like:
- Figma/Sketch for design
- https://www.appicon.co/ - Icon generator
- https://apetools.webprofusion.com/ - Asset generator
- Or hire a designer on Fiverr/99designs

## Design Specs

**Icon**: 
- 1024x1024px PNG
- No transparency for iOS
- 192x192px safe zone for Android adaptive

**Splash**:
- 1242x2436px (iPhone 13 Pro Max)
- Brand color background (#9333ea)
- Centered logo/text

**Adaptive Icon**:
- 1024x1024px PNG
- 66% safe zone (center circle)
- Background color: #9333ea
